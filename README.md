# My Claude Plugins Marketplace

Personal Claude Code plugin marketplace containing:

- **ship-it-mcp** — GitHub integration: list repos, commit, push, manage PRs
- **turo-mcp** — Turo car rental: bookings, calendar, earnings, gap analysis

## Install

In Claude Code → Settings → Plugins → Add Marketplace:
```
jjm7271985/my-claude-marketplace
```
